# Workflow - Check-In Seance

Utiliser ce workflow avant et apres les seances.

## Priorite

Avant de valider une seance, verifier qu'elle correspond toujours a l'objectif actuel et a la periode actuelle.

Si la seance prevue ne sert plus le bon objectif, l'ajuster.

## Avant Seance

Collecter si necessaire :

- sommeil
- fatigue
- douleurs
- stress
- derniere seance
- charge sport principal/running recente
- temps disponible
- materiel disponible

Decision :

- faire la seance prevue
- alleger
- remplacer
- raccourcir
- deplacer
- passer en recuperation active

## Apres Seance

Journaliser :

- date
- type de seance
- exercices
- series/reps/charges
- RPE
- energie
- douleurs
- notes techniques
- commentaire global

Mettre a jour :

- `data/workout_history.csv`
- `data/strength_progression.csv` si charges/perfs pertinentes
- `data/running_history.csv` si course
