# Architecture Sportive

## Regle D'alignement

Avant de choisir une structure Push/Pull/Leg/Upper, verifier :

- objectif actuel
- periode actuelle
- priorite sport principal ou physique
- fatigue recente
- materiel disponible

La structure sportive doit servir l'objectif en cours. Elle peut changer selon la periode.

## Briques Hebdomadaires

Construire les semaines avec les briques suivantes selon contexte :

- Push
- Pull
- Leg
- Upper
- isometrie / pliometrie
- mobilite
- etirements
- running
- seance specifique sport principal
- poids du corps maison si fatigue ou impossibilite salle

## Structure Standard Seance Salle

### Echauffement Abdos / Dos

- Extension dos : 3x10 a 20 kg
- Releve genou suspendu : 4x15
- Crunch poulie : 4x10 a 27 kg

### Corps De Seance

Adapter selon le type de jour : push, pull, leg, upper, specifique sport principal, force, hypertrophie ou deload.

## Couverture Musculaire Et Rotation Des Exercices

Sur un cycle de plusieurs semaines, ne pas repeter aveuglement exactement les memes angles si cela limite le developpement.

Programmer les exercices pour couvrir progressivement toutes les portions utiles des groupes musculaires, selon l'objectif en cours :

- pecs : incline, horizontal, adduction / ecartes
- dos : tirage vertical, tirage horizontal, tirage convergent ou unilateral
- epaules : developpe, deltoide lateral, arriere d'epaule / stabilisateurs
- jambes : quadriceps, ischios, fessiers, mollets, unilateral stable, chaine posterieure
- bras : biceps supination/incline, marteau/brachial, triceps extension verticale et poussee

Ne pas changer d'exercice juste pour changer. Garder les mouvements principaux assez longtemps pour mesurer la progression, puis faire tourner les variantes quand :

- une portion musculaire est sous-travaillee
- une machine est indisponible
- la progression stagne
- une douleur ou gene apparait
- la periode change
- le contexte sport principal impose moins de fatigue ou moins de risque

Priorite : couverture musculaire complete, progression mesurable, prevention blessures, transfert vers le sport principal.

### Fin De Seance

- Marche inclinee
- Mobilite si necessaire

## Exercices De Reference

Les exercices exacts doivent venir du profil et de l'historique de l'utilisateur. Les exemples ci-dessous servent seulement de base a adapter.

### Bas Du Corps

- Squat, presse ou variante unilateral stable
- Hip thrust ou pont fessier
- Souleve de terre roumain ou hip hinge equivalent
- Leg curl ou variante ischios
- Mollets
- Isometrie ou prehab si une zone est sensible

### Haut Du Corps

- Tirage vertical
- Tirage horizontal
- Developpe horizontal ou incline
- Developpe epaules si adapte
- Elevations laterales
- Face-pull ou travail scapulaire
- Bras selon objectif et temps disponible

## Specifique Sport Principal

Integrer :

- explosivite
- appuis
- vitesse laterale
- gainage
- prevention blessures
- epaules
- chevilles
- zones sensibles
- endurance specifique sport principal

Seance specifique sport principal possible :

- 10 min velo
- 10 min course
- musculation adaptee au sport principal
